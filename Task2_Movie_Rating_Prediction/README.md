# Movie Rating Prediction with Python

## CodSoft Data Science Internship - Task 2

### Project Objective

The objective of this project is to build a machine learning regression model that predicts movie ratings based on features such as genre, director, actors, duration, year, and votes.

---

## Dataset

**Dataset Name:** IMDb Movies India Dataset

### Features

- Name
- Year
- Duration
- Genre
- Rating (Target)
- Votes
- Director
- Actor 1
- Actor 2
- Actor 3

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Machine Learning Algorithm

- Random Forest Regressor

---

## Project Workflow

1. Import required libraries
2. Load the dataset
3. Explore the dataset
4. Handle missing values
5. Encode categorical variables
6. Clean numerical columns
7. Split the dataset into training and testing sets
8. Train the Random Forest Regressor
9. Predict movie ratings
10. Evaluate the model

---

## Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

---

## Result

The trained model predicts movie ratings based on movie characteristics such as genre, actors, director, duration, year, and votes.

---

## Conclusion

This project demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, prediction, and evaluation using a regression algorithm.
